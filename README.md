# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- 2 Player game (X and O)
- Win detection logic
- Draw condition handling
- Simple and clean UI

---

## ⚒️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 💡 About Project

This project is built to practice JavaScript logic, DOM manipulation, and event handling. It helps improve problem-solving skills in frontend development.
